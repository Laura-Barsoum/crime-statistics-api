About the Dataset

Data Source & Coverage

This API provides access to comprehensive crime statistics for all US states from 1960 to 2019 (60 years). The data is sourced from the FBI's Unified Crime Reporting (UCR) program and includes both violent crimes (murder, rape, robbery, assault) and property crimes (burglary, larceny, motor vehicle theft).

Data Source: CORGIS Datasets Project - State Crime Dataset
Original Source: U.S. Department of Justice and Federal Bureau of Investigation

Key Features:

  • Crime rates per 100,000 population (normalized)
  • Absolute crime counts (raw numbers)
  • Historical data spanning 60 years (1960-2019)
  • Complete coverage of all 50 US states plus territories
  • Breakdown by crime type (violent vs. property)
  • Population data for context
